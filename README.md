# site_new

This repository contains the React app in `react-app`.

## Start the app

1. Open a terminal in the project root.
2. Change into the app directory:

```powershell
cd react-app
```

3. Install dependencies the first time only:

```powershell
npm install
```

4. Start the development server:

```powershell
npm run dev
```

5. Open the local app in your browser:

```text
http://localhost:5173/
```

## Stop the app

Press `Ctrl+C` in the terminal that is running the dev server.

## Other useful commands

```powershell
npm run build
npm run preview
```
